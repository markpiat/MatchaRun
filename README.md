# MatchaRun

MatchaRun is a text-only starter React app for an adaptive running coaching product.

The app is designed to collect:

- Competition date
- Race distance
- Current race time
- Age
- Sex
- Injury or training constraints
- Training availability

It can then be extended to generate structured training plans containing easy runs, long runs, tempo runs, intervals, recovery guidance and progression rules.

## Run locally

```bash
npm install
npm run dev
```

## Text-only repository rules

This repository intentionally avoids binary files so it can be transferred cleanly through ChatGPT, Codex and GitHub web workflows.

Do not commit images, icons, archives, database files, build artefacts or cache files. Use inline SVG, CSS or text placeholders instead.
